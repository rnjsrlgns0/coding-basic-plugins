# 즉시 실행

아래 스킬들을 **순서대로** 실행하세요:

**1단계: 계획 수립**
```
Skill 도구 사용: skill: "development:task-planner"
```

**2단계: 브랜치 생성** (1단계 완료 후)
```
Skill 도구 사용: skill: "development:branch-manager"
```

**3단계: 작업 실행** (2단계 완료 후)
- 계획에 따라 작업 수행

**4단계: 메모리 업데이트** (작업 완료 후)
```
Skill 도구 사용: skill: "development:memory-bank-updater"
```

**5단계: 브랜치 병합** (4단계 완료 후)
```
Skill 도구 사용: skill: "development:branch-manager"
```

**각 스킬이 로딩되면 스킬의 지시사항을 즉시 따르세요.**
